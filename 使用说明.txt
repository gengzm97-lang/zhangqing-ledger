账清 · GPT 业务记账
====================

打开方式
--------
直接双击 index.html，即可使用，无需安装。

建议使用最新版 Microsoft Edge 或 Google Chrome。

主要功能
--------
1. 点击“记一笔”，记录客户及其来源、GPT Plus / 5x / 20x、成品号 / 代充、收入和成本。已有客户会自动带出来源，也可直接修改。
2. 每笔利润自动按“收入 - 成本”计算。
3. 经营看板可查看总收入、总利润、总成本，以及每项业务收入。
4. 客户管理会自动分析常购项目、交付偏好、平均充值间隔、常用下单时间和预计复购日。
5. 数据管理可导出 CSV 和完整 JSON 备份。
6. “个人支出”可记录餐饮、交通、购物、住房等日常消费，并按日期和分类汇总。
7. 部署为 HTTPS 网页并配置 Supabase 后，可安装到安卓桌面并在电脑、手机间自动同步。

个人支出与业务利润
------------------
个人支出使用独立账本，不会扣减 GPT 业务利润。这样既能了解个人消费，也不会影响经营分析。

重要提示
--------
默认情况下数据只保存在当前浏览器中；启用云同步后，数据也会保存到你配置的 Supabase 项目。
请定期进入“数据管理”导出完整备份。首次部署请阅读“PWA云同步部署说明.md”。
